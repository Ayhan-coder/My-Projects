Question 19:

Sum of Prime Pairs (5 points)

Please read important information first.

Description

In this problem, you are given a positive integer n. Your task is to find and print all unique pairs of prime numbers whose sum equals n.

A prime number is a positive integer greater than 1 that has no divisors other than 1 and itself.

User will input 1 number:

  • An integer n (that is bigger than 1)

In each line of the output will be:

  • A pair of prime numbers that sum up to n (check the examples for clarification).
  • The pair should appear in the reverse too. (i.e. if 2 5 is in the output, then 5 2 should be too.)
  • Order of the printing is important. If a b and c d are two pairs in the output, then
    • If c > a, c d is printed after a b
    • If c = a but d > b, c d is printed after a b
    • If c = a and d = b, meaning they are same pair, it's printed only once
  • Print nothing if no such pair exists.

Important: Input taking part is already handled.
