Question 5:

Is The Word Palindrome (5 points)

Please read important information first.

Description

In this problem, you are given a word with n characters. Your task is to check whether the word is palindrome or not.

When a word is read the same backwards as forwards, that word is called palindrome.

There are 2 inputs:

  • First input is a positive integer.
  • Second input is an english word

It is guaranteed that all letters are lowercase, you don't need to think about case sensitivity.

However output is case sensivite, please be careful about upper and lower cases when printing. Print "YES" or "NO"!

Depending on your input numbers, you need to print:

  • "YES" if the word is a palindrome.
  • "NO" if the word is not a palindrome.

Important: Input taking is already done by us and shouldn't be written by you.
