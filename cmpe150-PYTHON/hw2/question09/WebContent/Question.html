Question 9:

Weather Report (5 points)

Please read important information first.

Description

In this problem, you are given a week's temperatures in Celsius. You are expected to convert it to Fahrenheit and print the corresponding statement.

The formula for converting Celsius to Fahrenheit is:
F = (9/5) * C + 32
where F is the Fahrenheit value and C is the Celsius value.

Warning:Input taking is handled for you. The variable temperatures is a list of 7 integers representing the temperatures of a week (in order) in Celsius. You can assume this list will always contain positive integers.

  • If the Fahrenheit value for a day is greater than 80, print It's hot on day {day_number}.
  • If it is less than 60 (not inclusive), print It's cold on day {day_number}.
  • If it is in between, print It's warm on day {day_number}.
