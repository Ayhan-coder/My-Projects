Question 3:

List Permutations

Please read important information first.

Description

In this problem, you need to implement the permutations(my_list) function recursively. This function should assume my_list is a list of strings. Return a list of all the concatenated permutations of the strings of my_list.

For example:
if my_list = ["1", "2", "3"] then, permutations(my_list) should return ["123", "132", "213", "231", "312", "321"] Note that in the output printing part, we are already sorting the result of this function. Therefore, order of the resulting list does not matter.

Input taking part is handled for you, predefined variables are:

  • words: A non-empty list of non-empty words

Output printing part is handled for you as well, you only need to implement permutations(my_list) function recursively..

Following are true for every question:

  • Important Note 1: If you don't implement it recursively, your grade will be 0 even if your outputs are correct.
  • Important Note 2: Inside your recursive functions, you are not allowed to use:
    • Variables that have global keyword
    • Variables that are defined outside the recursive function (unless specified otherwise)
