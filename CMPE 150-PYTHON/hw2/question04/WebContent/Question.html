Description

In this question, you are tasked to find and print all the prime numbers smaller than a given positive integer n (A prime number is a positive integer greater than 1 that is not a product of two smaller positive integers.). They have to be printed in an ascending order.

User will enter 1 input:

At the end of the code you need to print:
