Fibonacci Sequence (5 points)

Please read important information first.

Description

In mathematics, the Fibonacci sequence is a sequence in which each number is the sum of the two preceding ones. Numbers that are part of the Fibonacci sequence are known as Fibonacci numbers. You are tasked to write a code that finds the n'th fibonacci number for any given n.

More robustly, let F(n) denote the n'th fibonacci number. Then F(n) = F(n-1) + F(n-2)

Assume F(0) = 0, F(1) = 1

User will enter 1 input:

  • A positive integer n

At the end of the code you need to print:

  • n'th fibonacci number.

Important: Input taking and output display is already done by us and shouldn't be written by you.
