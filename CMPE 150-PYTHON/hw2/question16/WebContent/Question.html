Question 16:

How long is a sentence? (5 points)

Please read important information first.

Description

In this problem, you are expected to find the number of words and letters inside a given sentence.

User will enter 1 input, which will be the sentence itself. You can safely assume that the sentence only consists of letters,
empty spaces, and a dot (.) at the end. The sentence will always end with a dot (.).

Depending on the input, you need to print:

  • The number of words inside the sentence (as first output)
  • The number of letters inside the sentence (as second output)

------------------------------------------------------------------

Check the examples for further clarification. Keep in mind that we will be grading your code not just based on these examples, but other cases as well, so try to write code which can handle all possible cases.

Warning: You are not allowed to use any imports and any topics that haven't been covered this semester.
